Информация
----------
Библиотека mem.dll является базовой для остальных проектов движка и должна собираться первой.

Описание
--------
Для выделения памяти необходимо использовать макрос mem_alloc, а для освобождения - функцию mem_free.
Они аналогичны функциям из C.
Например,
int *p = (int*)mem_alloc(sizeof(int));
mem_free(p);

Подключение
-----------
В исходниках вашего проекта нужно подключить заголовочный файл raw.h из папки src\cmdk\mem
В настройках вашего проекта для MSVC2008 необходимо прописать путь и саму библиотеку импорта mem.lib.
